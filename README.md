PASSWORD GENERATOR

This program is deisigned to create a randomly generated password within certain criteria provided by the user.

INSTRUCTION

Operation of this program is straightforword. All that is required is to click the 'Generate' button on the site and follow and prompts given to the user. Your resulting password will appear on-screen.

OPERATION

This program runs on javascript with a multitude of loops and functions. Each user input is assigned to a variable and said variables are used to determine which loop to run, and how long to run it. Each loop contains a differing collection of the central randomness functions to assign the correct characters to every password. Certain validation checks are in place to assure the user has provided workable inputs.
