PASSWORD GENERATOR

This program is deisigned to create a password within certain criteria provided by the user

INSTRUCTION

Operation of this program is straightword. All that is required is to click the 'Generate' button on the site and follow and prompts given to the user

OPERATION

This program runs on javascript with a multitude of loops. Each user input is assigned to a variable and said variables are used to determine which loop to run, and how long to run it.
